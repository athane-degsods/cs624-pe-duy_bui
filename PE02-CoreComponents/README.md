# 1. Input
The application uses two input streams: a static array of the MSCS course curriculum and a user input for their favorite course. The static array is defined within the application, while the user input is captured through a TextInput component.

# 2. Process
A React Native functional component is used to render the application with ScrollView, Image, TextInput, and Text components. The user can input their favorite course, which is stored in the component's state using the useState hook. The application displays the static course curriculum and the user's favorite course. 

# 3. Output
Stylesheet is implemented to provide a consistent look and feel across the application. The ScrollView allows for vertical scrolling of the content. The Image component displays an icon, and the TextInput component captures user input. The application dynamically updates the displayed favorite course based on user input.
